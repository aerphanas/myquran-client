# my quran client

my quran client api

in order to use this library you need sbcl and quicklisp installed

## how to use
- download/clone this repo
- go to src folder
- run ```sbcl --load main.lisp```

## function
- cari-kota: string -> list
- jadwal-hariini: number -> list
- jadwal-bulanini: number -> list
- tafsir: number -> list
